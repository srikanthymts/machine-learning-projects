# Machine Learning Projects for Students

Machine learning projects help students understand how artificial intelligence models learn from data and make predictions. These projects allow learners to build practical applications and improve their AI skills.

## Beginner Machine Learning Projects

### Iris Flower Classification
A classification model that predicts iris flower species using petal and sepal measurements.

### House Price Prediction
A regression model that predicts house prices using dataset features like size and location.

### Titanic Survival Prediction
A classification project that predicts whether passengers survived the Titanic disaster.

### Spam Email Detection
A Natural Language Processing (NLP) project that classifies emails as spam or not spam.

### Handwritten Digit Recognition
A neural network project using the MNIST dataset to recognize digits.
